Font Awesome使用手册

learn
  • Font Awesome

介绍

Font Awesome 是一套完美的图标字体,主要目的是和 Bootstrap 搭配使用。Font Awesome提供可缩放的矢量图标，您可以使用CSS所提供的所有特性对它们进行更改，包括：大小、颜色、阴影或者其它任何支持的效果。（如果需要支持IE7，那真的十分抱歉。Font Awesome从4.1.0版本开始不再支持IE7， 但更早的版本仍然支持。）

使用方法

直接下载

直接从官网下载Font Awesome，然后在网页头部引用即可在网页中使用Font Awesome提供的图标。

引用CDN服务

直接在网页头部引用CDN。

使用指南

基本图标

Font Awesome图标使用在几乎任何地方，只需要使用CSS前缀 fa ，再加上图标名称。 Font Awesome是为使用内联元素而设计的。我们通常更喜欢使用 ，因为它更简洁。 但实际上使用 才能更加语义化。

照相机

大图标

使用 fa-lg (33%递增)、fa-2x、 fa-3x、fa-4x，或者 fa-5x 类 来放大图标。如果以上大小都不能满足要求可以使用font-size来设置大小。

fa-2x

列表

使用 fa-ul 和 fa-li 便可以简单的将无序列表的默认符号替换掉。

动画

使用 fa-spin 类来使任意图标旋转，现在您还可以使用 fa-pulse 来使其进行8方位旋转。尤其适合 fa-spinner、fa-refresh 和 fa-cog 。





旋转与翻转

使用 fa-rotate- 和 fa-flip- 类可以对图标进行任意旋转和翻转。

固定宽度

使用 fa-fw 可以将图标设置为一个固定宽度。主要用于不同宽度图标无法对齐的情况。 尤其在列表或导航时起到重要作用。

边框与对齐

使用 fa-border 以及 pull-right 或 pull-left 可以轻易构造出引用的特殊效果。
